# WeekDesk

WeekDesk 是一款 Windows 桌面日历待办工具，支持桌面穿透、快速编辑、主题切换和本地数据管理。

这个仓库只用于展示产品介绍、下载入口和版本说明，不包含 WeekDesk 的应用源代码。

## 下载

请到 GitHub Releases 下载最新版安装包：

https://github.com/YuWan-030/WeekDesk/releases

当前准备发布版本：`0.1.4.7`

安装包文件：

`WeekDesk Setup 0.1.4.7.exe`

## 功能概览

- 桌面日历视图
- 每日待办记录
- 编辑模式 / 穿透模式切换
- 编辑模式下可滚动查看溢出的待办内容
- 免费版和 Pro 版功能标识
- Pro：主题切换、自定义透明度、自定义休息日、自定义快捷键、数据导入导出、桌面层挂载实验功能、30 天未完成待办自动顺延
